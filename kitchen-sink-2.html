    Scroll Effects

    Available Classes: Add .navbar-shrink for height reduction on scroll, .navbar-transparent for hero overlays that become solid on scroll, or .navbar-autohide for hide-on-scroll-down behavior. Apply .scrolled class via JavaScript when page scrolls. 
    // Example scroll handler
window.addEventListener('scroll', () => {
  const navbar = document.querySelector('.navbar');
  if (window.scrollY > 50) {
    navbar.classList.add('scrolled');
  } else {
    navbar.classList.remove('scrolled');
  }
});
